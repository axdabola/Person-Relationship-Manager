# Person-Relationship-Manager
C++ Console App, demonstrating use of vectors in a similar way of Collection&lt;T> in C#. 
<br> Lang: pt-pt + en-us

# Files:
<ul>
  <li>main.cpp -> main function</li>
  <li>datavector.hpp -> class PessoaCol (namespace datavector)</li>
    <ul>
      <li>This file contains the vector inside the class, which can be used with the sampled funcs. in the file, like add a pessoa_t obj. to the vector</li>
    </ul>
   <li>defcls.hpp -> class Enums && Funcs (namespace defcls)</li>
   <li>defdata.hpp -> class Pessoa (typedef pessoa_t) && VectorData (typedef vcdata_t) (namespace defdata)</li>
   <li>scrman.hpp -> funcs. for Console (cmd) based operations (namespace scrman)</li>
</ul>
